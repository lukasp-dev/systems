# Variables in Memory

```cpp
#include <iostream>

int main(){
    int a = 0;
    int b = 3;
    int c = a+b;
    double d = 2.5;

    std::cout << "a:" << (long)&a << std::endl;
    std::cout << "b:" << (long)&b << std::endl;
    std::cout << "c:" << (long)&c << std::endl;
    std::cout << "d:" << (long)&d << std::endl;

    return 0;
}
```

기본 변수 타입은 그 크기가 컴퓨터 환경마다 달라질 수 있으므로
- `int` 대신에 `int8_t`, `int32_t` 처럼 명시해서 짜는게 더 좋다. 참고로 뒤에 붙은 숫자는 bit 수임.

```cpp
#include <array>
#include <cstdint>
#include <iostream>

int main(){
    uint64_t ui8;
    float f4;
    std::array<uint8_t, 5> uia5;

    uint64_t* ui64ptr = &ui8;
    std:: cout << sizeof(ui64ptr) << std::endl;
    std;:cout << (uint64_t)ui64ptr << std::endl;
}
```

### static_assert
컴파일 할 때 조건을 검사해서, 조건에 맞지 않으면 빌드를 중단시키는 안전장치.

- `static_assert(bool-constexpr, unevaluated_string)`

- `static_assert(bool-constexpr)`

- `static_assert(bool-constexpr, constant-expresstion)`

예시 1
```cpp
#include <iostream>

int main(){
    static_asert(sizeof(int) == 4, "int should be d4 bytes.\n");
    int a = 0;

    std::cout << sizeof(int) << std::endl;
    std::cout << sizeof(a) << std::endl; 

    return 0;
}
```


예시 2
```cpp
#include <iostream>

constexpr = int MAX_ID_LENGTH = 10;

struct User{
    char id_buffer[MAX_ID_LENGTH];
};

int main(){
    static_assert(sizeof(User::id_buffer) == MAX_ID_LENGTH, "Error: 유저 버퍼 크기가 설정값과 다르다.\n");

    // 4. 런타임에 바꾸려고 하면?
    // MAX_ID_LENGTH = 20; // ❌ 당연히 컴파일 에러! 런타임에 절대 못 바꿉니다.
    return 0;
}
```

### struct 와 class 의 memory layout

```cpp
#include <iostream>
#include <cstdint>

struct ST {
    int64_t a;  //8 bytes
    int32_t b;   //4 bytes
    int16_t c; //2 bytes
};

class Cat{
public:
    void printCat(); // function 은 class 의 사이즈에 관여하지 않는다.

private:
    int32_t age;
    float weight;
};

int main(){
    std::cout << sizeof(ST) << "\n"; //16
    std::cout << sizeof(Cat) << "\n"; //8
    
    Cat cat1;
    Cat cat2;

    Cat* cat_ptr = &cat1;
    std::cout << sizeof(cat_ptr) << "\n"; //8

    return 0;
}
```
