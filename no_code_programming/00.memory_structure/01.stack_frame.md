# Stack Frame
Stack Memory 는 변수별로 쌓여가는게 아니라 Stack Frame 단위로 쌓인다.
stack frame 은 모든게 컴파일 시간에 결정된다.

(스택 프레임 설명과 사진)

```cpp
#include <iostream> 

class Cat {
public:
    Cat(){
        m_age = 1;
    };
    ~Cat();

    void addAge(int arg){
        m_age += arg;
    };
private:
    int m_age;
};

int main(){
    Cat cat;
    cat.addAge(10);

    return 0;
}
```

(stack 메모리 레이아웃 during this code)
