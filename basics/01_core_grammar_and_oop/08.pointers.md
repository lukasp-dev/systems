\* is dereferencing operator

**Pointer v. Reference**

The distinction between a pointer and a reference is more about semantics and safety guarantees than what the CPU actually sees. At the machine code level, both usually boil down to a memory address stored in a register.

| Feature | Pointer (`int* ptr`) | Reference (`int& ref`) |
| :--- | :--- | :--- |
| **Identity** | A variable that **holds** a memory address. | An **alias** (nickname) for an existing variable. |
| **Initialization** | Can be declared without an initial value (`int* p;`). | **Must** be initialized upon declaration. |
| **Reassignment** | Can be changed to point to a different address later. | **Cannot be rebound.** It stays with its original object. |
| **Nullability** | Can be `nullptr`. | Cannot be `null` (legally). |
| **Indirection** | Requires dereferencing (`*ptr`) to access value. | Accessed directly like a normal variable (`ref`). |

### Modern pointer safety checklist

### 1) Prefer owning abstractions
- Raw pointer (`T*`) should usually mean non-owning observation.
- Owning memory should usually be `std::unique_ptr<T>` (or `std::shared_ptr<T>` when shared ownership is required).

### 2) Null and dangling are different bugs
- Null pointer: address is intentionally empty (`nullptr`).
- Dangling pointer: address is non-null but object lifetime already ended.
- Dangling is more dangerous because checks like `if (p)` still pass.

### 3) Pointer invalidation cases
- `std::vector` reallocation invalidates pointers/references/iterators to its elements.
- Deleting object invalidates all pointers/references to it.
- Returning address of local variable creates immediate dangling pointer.

### 4) Use `std::span` for view semantics (C++20)
For "pointer + length" APIs, prefer `std::span<T>`:

```cpp
void process(std::span<const int> data);
```

This encodes bounds and keeps call sites clean.

### Full source (08.pointers.cpp)

```cpp
#include <iostream>

using namespace std;

void change(int *x){
    *x = 10;
}

//method overloading
void change(int &x){
    x = 20;
}

int main(){
    int x = 5;
    change(&x);
    
    cout << x << "\n"; //10 

    change(x);

    cout << x << "\n"; //20

    return 0;
}
```
