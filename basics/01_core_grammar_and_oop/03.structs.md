### Structs
The main difference between structs and classes are stemming from the access modifiers.

In the structs, the members are, by default, **public**.
In the classes, however, the members are **private** by default.

- possible access modifiers
    - public
    - private 
    - protected

#### What is PODs?
In C++, POD stands for Plain Old Data, a category of simple types (like C-style structs) that lack complex object-oriented features such as user-defined constructors, destructors, or virtual functions. This simplicity guarantees da predictable and consistent memory layout, similar to that used by the C programming language, which is crucial for low-level memory manipulation and interoperability.
```cpp
struct Point {
    int x;
    int y;
}
```
